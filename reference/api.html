Exploring the API¶

A quick way to see the API in action:

  1. Add Packages\Default (Preferences | Browse Packages…) to your project.
  2. CTRL + SHIFT + F
  3. Enter *.py in the In Files: field
  4. Check Use Buffer option
  5. Search API name
  6. F4
  7. Study relevant source code
